# 🌐 Bootstrap 3-Page Website

This is a simple and beginner-friendly **3-page website** built using **HTML and Bootstrap**. It includes a responsive design and basic UI components for a modern look.

---

## 📌 Features

* 🏠 Responsive Homepage
* 🔐 Login Page with form
* 📝 Registration Page with form
* 📱 Mobile-friendly design using Bootstrap
* 🎨 Clean and simple UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5

---

## 📁 Project Structure

```bash
/project-folder
│── index.html        # Homepage
│── login.html        # Login page
│── register.html     # Registration page
│── style.css         # Custom styles (optional)
│── /assets           # Images or other assets (optional)
```

---

## 📄 Pages Overview

### 🏠 Homepage (`index.html`)

* Navigation bar
* Welcome section (Hero section)
* Links to Login and Registration pages

### 🔐 Login Page (`login.html`)

* Email/Username input
* Password input
* Login button
* Link to registration page

### 📝 Registration Page (`register.html`)

* Name input
* Email input
* Password input
* Confirm password field
* Register button

---

## ▶️ How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/palak/bootstrap-project.git
   ```

2. Open the project folder

3. Open `index.html` in any web browser

---

## 🌍 Bootstrap CDN

This project uses Bootstrap via CDN. Make sure the following link is included in your HTML files:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
```
